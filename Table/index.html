Tabel HTML

  • Tabel HTML itu digunakan untuk mengatur data ke dalam baris dan kolom
  • Dalam Tabel HTML itu terdiri dari kolom dan baris
  • Baris itu yang horizontal (row)
  • Kolom itu yang vertikal
  • Untuk membuat tabel di HTML ini kita bisa membuatnya didalam tag
Contohnya :
Header 1 Header 2
Baris 1, Kolom 1 Baris 1, Kolom 2
Baris 2, Kolom 1 Baris 2, Kolom 2
Baris 3, Kolom 1 baris 3, Kolom 2


Kali ini kita akan melihat penggunaan caption pada tabel

contohnya :

List Pembelian Barang
Nama Barang Harga Barang
